# APP VUE 2.6.12 WEBPACK 5 2021

## Description

Development in a productive environment, updating of corresponding package dependencies for continuous compilation.

- Vue 2.6.12
- Axios
- Webpack 5
- CSS 
- SCSS
- Babel
- Moment
- Maps
- Git
- Express
- NodeJS
- Https production

## Install 

```bash
npm install 
```
or
```
yarn install
```
## Developing 

```
npm start
```

## Build

```
npm run build
```
